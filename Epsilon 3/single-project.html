Responsive Design

The webpage is responsive, i.e, it is coded to adjust as per and cater to, screens of different widths. The content structure of the webpage changes depending on the device it is viewed on. This was done with CSS grids and bootstrap classes to achieve a website which feels native to whatever device it may be running on. For eg. At mobile viewports, the navigation bar triggers a collapsable menu.
